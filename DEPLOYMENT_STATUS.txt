  This creates:

  STEP 2: Deploy to Appwrite Sites (5 minutes)
  ═══════════════════════════════════════════════════════════
  
  1. Go to: https://cloud.appwrite.io/console/project-68de87060019a1ca2b8b/sites
  2. Click "Create Site"
  3. Upload dist/ folder (or connect GitHub)
  4. Add environment variables from .env.production
  5. Click "Deploy"
  
  Your site will be live at: https://[your-app].appwrite.global
  

  STEP 3: Create Admin User (2 minutes)
  ═══════════════════════════════════════════════════════════
  
  1. Appwrite Console → Auth → Create User
  2. Email: admin@soukel-sayarat.com
  3. Copy User ID
  4. Databases → users → Add Document
  5. Use User ID, set role: "admin"
  

  STEP 4: Test & Launch (3 minutes)
  ═══════════════════════════════════════════════════════════
  
  1. Visit your Appwrite Sites URL
  2. Test login/signup
  3. Test product features
  4. Launch! 🚀
